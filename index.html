1. Villageois

Les villageois logés dans le bâtiment à villageois sont à votre disposition mais ces accès restent reglementés pour leurs sécurités. Nous vous demandons de respecter les règles suivantes :

  • Les tuer et les taper est formellement interdit
  • Casser leurs blocs de métier pour le replacer est formellement interdit
  • Frapper ou tuer les golems est formellement interdit
